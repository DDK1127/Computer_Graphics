# 作業說明
計算機圖學 (Computer Graphics) Homework
**Render a 3D Mesh**

## 編譯／執行環境

- OS：Windows 11
- IDE：Visual Studio
- C++ with OpenGL

## 如何執行

1. 安裝相關程式語言與套件。
2. 使用 Visual Studio Code 開啟專案資料夾。
3. 在終端機輸入指令執行程式，例如：
   - `python main.py`
   - `g++ main.cpp -o main && ./main`
   - `javac Main.java && java Main`
